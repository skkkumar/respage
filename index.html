Post processing comparison


We can observe that due to the post processing step there is a performance raise. Post processing step is to discard the frames which has 80% of the keypoints to be detected as anomolies. In the output videos we can notice the frames with green plot of anomoly as discarded frames. The red and the pink can be considered to be included. As I showed in the ground truth page, by including the stabilization data the result improved. But still, I included 10% of the whole data to be stabilization data and I am not sure how much percent is really necessary.
